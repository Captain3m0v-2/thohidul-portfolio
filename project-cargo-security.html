FEATURED PROJECT

IoT-Based Cargo Security & Health Monitoring System

Leveraging Fuzzy Logic and Embedded Systems for Real-time Theft Prevention and Autonomous Health Monitoring.

IoT & Embedded Systems Fuzzy Logic IEEE Published (ICCCNT 2023)

The Challenge & Objective

The objective was twofold: to design a robust, two-step anti-theft system for cargo vehicles, and separately, to create an autonomous, reliable patient health monitoring system. Traditional systems often lack real-time decision-making capability, a gap we addressed using algorithmic control.

Solution & Architecture

The core innovation was the implementation of a **Fuzzy Logic Inference System** (FIS). This approach allowed the system to make nuanced, autonomous decisions based on multiple, varying input parameters (like vital signs or sensor data), vastly improving reliability over simple threshold-based systems.

  • **Cargo Security:** Designed a two-step system for continuous, real-time tracking and theft prevention using geo-fencing and physical deterrents.
  • **Health Monitoring:** Implemented autonomous patient monitoring through IoT, making reliable decisions based on input from various sensors.
